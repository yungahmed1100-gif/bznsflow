## 🎬 Video Setup
The hero background video is **not included in this repository** (too large for GitHub).
To make the hero video work:
1. Name your video file exactly: `hero-bg.mp4`
2. Place it in the **root folder** (same level as `index.html`)
3. The code is already wired — it will work immediately
---
## 🖥 Local Development
### Option 1 — VS Code Live Server (Recommended)
1. Open the folder in VS Code
2. Install the **Live Server** extension by Ritwick Dey
3. Right-click `index.html` → **Open with Live Server**
4. Opens at `http://127.0.0.1:5500`
### Option 2 — Simple Browser Open
Just double-click `index.html` — works for basic preview (video may not autoplay on some browsers without a server)
---
## 🚀 Deployment (Netlify)
1. Go to [netlify.com](https://netlify.com) → **Add new site** → **Deploy manually**
2. Drag your entire project folder (including `hero-bg.mp4`) into the drop zone
3. Done — Netlify gives you a live URL instantly
Or connect directly to this GitHub repo for **auto-deploy on every push**.
